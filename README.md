
# Human Resource Management System (HRMS)

## Project Overview

This project is a **Console-Based Human Resource Management System (HRMS)** developed using **C++**.

The application demonstrates the core logic of a Human Resource Management System, including employee registration, authentication, attendance management, leave management, and salary management.

This project was developed as a prototype for a first-year hackathon.

---

## Technologies Used

- C++
- Structures
- Functions
- Arrays
- Object-Oriented Programming (Basic)
- Git & GitHub

---

## Features

### Authentication
- User Sign Up
- User Sign In
- Default Admin Login
- Employee Login

### Employee Module
- View Profile
- Daily Attendance (Check-In)
- Apply for Leave
- View Leave Status
- View Salary Details

### Admin Module
- View All Employees
- View Attendance Records
- Approve / Reject Leave Requests
- Update Employee Salary
